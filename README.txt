# Bollywood Mania Interactive Game App - Updated Version

## Changes in this version
1. Reveal Answer is now available anytime. You do not need to wait for 60 seconds.
2. Emoji clues are extracted directly from the provided PowerPoint and stored in the `assets` folder.
3. Rapid Fire has updated category-style questions:
   - Sports movies
   - Thriller / suspense movies
   - Comedy movies
   - Shah Rukh Khan movies
   - Akshay Kumar movies

## How to run
1. Unzip the folder.
2. Open `index.html` in Chrome or Edge.
3. Use Next / Prev to move like a presentation.
4. Timer auto-starts on question slides.
5. Use Pause / Start / Reset from the right-side HUD.
6. Use Reveal Answer anytime.
7. Use score buttons for each team: +10, +5, -5.

## Keyboard shortcuts
- Right Arrow: Next slide
- Left Arrow: Previous slide
- Space: Pause / Start timer
- R: Reset timer
- A: Reveal answer
